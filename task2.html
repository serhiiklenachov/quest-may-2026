Скриня №2: Дешифрування алгоритму

Змінна X = 15
Початок циклу: Повторити 3 рази
X = X + 9
Кінець циклу
Якщо X ділиться на 2 без остачі (парне):
X = X / 2
X = X * 3
Інакше:
X = X + 100
Якщо X > 50 ТА X < 70, тоді:
X = X + 4
Інакше:
X = X - 3

Виконай цей алгоритм крок за кроком та введи фінальне значення змінної X, щоб зламати скриню.

Помилка обчислень, спробуй ще раз!
